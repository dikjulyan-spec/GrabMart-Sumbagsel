Login GrabMerchant — Panduan Lengkap

Sumber: help.grab.com/merchant/id-id · gclogistik.com (verified)

📱 Login via GrabMerchant App

1
Download GrabMerchant App
Cari "GrabMerchant" di Google Play Store atau Apple App Store. App ini menggantikan GrabFood Merchant dengan fitur lengkap untuk mengelola GrabMart, GrabFood, dan OVO.
2
Buka App → Tap "Masuk"
Di halaman utama, tap tombol Masuk. Tersedia dua pilihan: Masuk dengan Google (Gmail terdaftar) atau masuk dengan Email + Password.
3
Pilih Akun Google / Masukkan Email
Untuk login Google: pilih akun Gmail yang terdaftar saat onboarding. Untuk login email: masukkan email dan password GrabMerchant (min. 8 karakter, huruf besar+kecil+angka).
4
Verifikasi OTP (jika diminta)
Grab mengirim kode OTP 4–6 digit via SMS ke nomor HP terdaftar. Masukkan kode dalam batas waktu yang tertera. Pastikan sinyal HP bagus.
5
Pilih Outlet (jika multi-outlet)
Jika punya beberapa outlet, pilih outlet yang akan dikelola. Anda bisa berpindah outlet kapan saja dari dalam app.
6
✅ Dashboard Terbuka
Selamat! Dashboard menampilkan: Pesanan Masuk, Status Toko (Buka/Tutup), dan Omzet Hari Ini secara real-time.

⚠️ Troubleshooting Login — Solusi Masalah Umum

❌ OTP Tidak Masuk
Cek koneksi internet. Pastikan nomor HP aktif dan tidak blokir SMS premium. Tunggu 60 detik, minta kirim ulang. Kalau masih gagal, hubungi support@grab.com
❌ Login Gagal / Akun Terkunci
Terjadi setelah beberapa kali salah password. Gunakan fitur "Lupa Kata Sandi" untuk reset via email. Kata sandi sementara dikirim ke inbox.
❌ Akun Tidak Ditemukan
Pastikan email sama persis dengan saat onboarding. Jika pindah email/HP, hubungi Grab Merchant Support untuk update data akun.
❌ App Crash / Error
Update GrabMerchant ke versi terbaru di Play Store / App Store. Jika masih error, clear cache app (Settings → App → GrabMerchant → Clear Cache) atau reinstall.
📞 Kontak support: support@grab.com | Hotline: +62 21 2350 7042 | GrabMerchant Partners: +62 21 8064 8787

Akses penuh: order, laporan keuangan, pengaturan akun, manajemen staf

Modul 01
Setup Profil Toko
  • Login → tap ikon Profil di pojok kanan atas → "Edit Profil Toko"
  • Upload foto cover toko resolusi minimal 800×600px (JPG/PNG)
  • Isi deskripsi toko yang menarik — highlight produk unggulan
  • Set Minimum Order Value (MOV) dan radius pengiriman
  • Atur jam operasional sesuai jadwal aktual bisnis Anda
💡 Tips: Foto toko cerah + produk rapi = CTR naik hingga 40%
⏰
Modul 02
Jam Operasional & Holiday Mode
  • Pengaturan → "Jam Operasional" → toggle ON/OFF per hari
  • Set waktu buka/tutup untuk setiap hari dalam seminggu
  • Gunakan Holiday Mode untuk tutup sementara tanpa menghapus toko
  • Aktifkan Auto-Close agar toko otomatis tutup di luar jam operasional
💡 Tips: Jam operasional konsisten = kepercayaan pelanggan dan algoritma Grab lebih tinggi
📍
Modul 03
Verifikasi Lokasi GPS
  • Profil → "Lokasi Toko" → pastikan pin di peta tepat sesuai alamat fisik
  • Drag pin jika posisi tidak akurat, konfirmasi dengan foto Google Maps
  • Atur area pickup driver yang mudah diakses (parkiran/gerbang)
  • Radius pengiriman default 10km, bisa dikustomisasi
⚠️ Penting: GPS tidak akurat → driver kesulitan → cancel rate naik → rating turun
🔔
Modul 04
Setup Notifikasi Order
  • Aktifkan notifikasi push untuk GrabMerchant di pengaturan HP
  • Set volume notifikasi KERAS agar tidak ada order terlewat
  • Aktifkan "Printer Receipt" jika pakai printer termal struk
  • Pertimbangkan tablet/HP kedua khusus di kasir untuk terima order
  • Fitur Perencana Orderan tersedia untuk ambil order pada jadwal tertentu
💡 Tips: Response time <3 menit = prioritas tampil di halaman utama Grab

Modul 01
Upload Produk Baru

  • Isi deskripsi: berat, varian, tanggal kadaluarsa (produk makanan)
  • Set harga jual dan harga coret (asli sebelum diskon)
  • Input stok awal → aktifkan toggle "Tersedia" → produk langsung live
💡 SEO: "Beras Premium 5kg Pulen" lebih mudah ditemukan dibanding hanya "Beras"

Modul 02
Bulk Upload via Excel

  • Isi data produk di Excel: nama, deskripsi, harga, kategori, stok
  • Upload kembali file Excel yang sudah diisi → sistem proses otomatis
  • Cocok untuk toko dengan >10 produk — hemat waktu signifikan
💡 Tips: Toko dengan 50+ produk aktif mendapat lebih banyak exposure di halaman kategori

Manajemen Stok Real-Time
  • Update stok setiap hari sebelum toko buka untuk hindari oversell
  • Gunakan "Mark as Unavailable" untuk produk habis sementara (jangan hapus)
  • Set stok minimum alert — notifikasi otomatis saat stok mendekati habis
  • Aktifkan "Auto Pause" jika semua produk habis — toko otomatis pause

💡 Tips: Kategori yang terstruktur rapi meningkatkan waktu kunjungan dan konversi pelanggan

Modul 01
Terima & Konfirmasi Order
  • Notifikasi masuk → HP berdering → tap "Terima Pesanan" dalam 3 menit
  • Cek detail: nama item, jumlah, catatan khusus, dan alamat pengiriman
  • Jika item habis: gunakan "Edit Order" → hapus item → konfirmasi ke pelanggan
  • Siapkan item → tap "Siap Diambil Driver" untuk panggil driver
⏱️ Target: Acceptance Rate >90% + Preparation Time <10 menit = performa optimal
📦
Modul 02
Pengemasan yang Benar
  • Kemas rapi dan aman — kantong plastik tebal atau kardus untuk item berat
  • Beri label nama pelanggan + nomor order di setiap kemasan
  • Produk dingin/beku: styrofoam + ice gel untuk jaga suhu
  • Produk cair: seal ekstra + plastik kedap udara — hindari tumpah
  • Pisahkan makanan dari produk kimia/pembersih dalam satu order
💡 Tips: Kemasan rapi = review bintang 5 dari pelanggan
🚴
Modul 03
Koordinasi dengan Driver
  • Setelah "Siap Diambil", sistem cari driver terdekat otomatis
  • Pantau nama driver dan estimasi tiba di halaman pesanan aktif
  • Siapkan item di area pickup yang mudah diakses driver
  • Jika driver lama: tap "Hubungi Driver" untuk konfirmasi posisi
  • Serahkan pesanan → tap "Pesanan Diserahkan ke Driver"
💡 Tips: Area pickup jelas dan mudah diakses = waktu tunggu driver lebih singkat
↩️
Modul 04
Komplain & Refund
  • Riwayat Pesanan → order bermasalah → "Ada Masalah" → pilih jenis masalah
  • Upload foto bukti jika diperlukan untuk klaim refund ke Grab
  • Komunikasikan solusi ke pelanggan via fitur chat in-app
  • Ajukan dispute jika refund tidak adil: Keuangan → Dispute
  • Proses dispute 7–14 hari kerja oleh tim Grab
💡 Tips: Dokumentasikan setiap pesanan dengan foto sebelum diserahkan ke driver
