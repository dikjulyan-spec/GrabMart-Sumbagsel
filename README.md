# GrabMart Sumbagsel v2

Struktur proyek hasil upgrade UI:

- `index.html` landing page baru yang lebih ringkas dan conversion-oriented.
- `admin.html` CMS admin existing tetap disertakan.
- `assets/css/styles.css` styling utama.
- `assets/js/main.js` logic interaktif dan integrasi Google Apps Script.
- `assets/images/` logo dan ilustrasi.
- `robots.txt`, `sitemap.xml`, `CNAME`, dan file verifikasi Google tetap disertakan.

## Catatan deploy

Upload seluruh isi folder ini ke GitHub repository / hosting yang sama.
Pastikan `CNAME` tetap di root repository.
